List of Apache Camel extensions for Quarkus

As of Camel Quarkus 1.0.0-M3 Camel artifacts listed below are supported on Quarkus.

In case you are missing some Camel feature in the list:

+ * Upvote an existing issue or create a new one so that we can better priorize our work. * You may also want to try to add the extension yourself following our Contributor guide. * You may try your luck using the given camel component on Quarkus directly (without an extension). Most probably it will work in the JVM mode and fail in the native mode. Do not hesitate to report any issues you encounter.
